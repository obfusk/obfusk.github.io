m - minimalistic media manager
m @ GitHub
m-gui @ GitHub

m keeps track of which files you've played (or are still playing) and thus allows you to easily continue playing the next file (using vlc or mpv).

And for situations where you prefer simple keybindings to typing on the command-line, there's the m-gui wrapper.

More information and examples: → m README @ GitHub → m-gui README @ GitHub
